## Stack Memory

The **Stack** is a specialized, contiguous segment of computer memory **automatically allocated** by the operating system to manage execution threads, function calls, and local variables.

Whenever a function is invoked in a programming language (like C, C++, or Java), a block of memory called a **Stack Frame** (or Activation Record) is pushed onto the runtime stack.
#### Variable Declaration

- Whenever a variable is declared, it's stored in the Stack
