## Linear Search

**Linear search** is a simple searching algorithm that **checks every element in a data collection one by one** from start to finish until it finds the target value or reaches the end.

## Efficiency
#### Time Complexity

- The time complexity of a linear search algorithm is ==**O(n)**== in the worst and average cases
- This means that the execution time **grows proportionally** with the **size** of the dataset (n)
