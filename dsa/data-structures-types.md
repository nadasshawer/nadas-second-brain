## Types of Data Structures

#### Physical Data Structures

Define how data is physically stored and arranged in memory. They actually hold the data physically in the memory.

- [Arrays](arrays.md)
- [Linked Lists](linked-lists)
#### Logical Data Structures

Define how the data can be utilized. Logical data structures are implemented using physical data structures (either array or linked list).

**Linear:**
- [Stack](stack-memory.md) → LIFO (Last In First Out)
- [Queues](queues) → FIFO (First In First Out)

**Non-linear:**
- [Trees](trees) → hierarchy
- [Graph](graph) → collection of nodes and connection between them

**Tabular:**
- [Hashing](hashing)
