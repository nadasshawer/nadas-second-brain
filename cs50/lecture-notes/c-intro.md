## Lecture 1
### Introduction
##### Types of Code

- **Source code** — code we the humans write
- **Machine code** — binary code the computer understands
##### Compilers

- Source code —> compiler —> machine code
##### 0s & 1s

- Any value that is `0` is interpreted as **false**
- Any value that is `1` or a non-zero is interpreted as **true**
#### Loops

##### Common Types of Loops

- `while` loop
	- loops while a condition is **true**
	- `while(true)` or `while(1)` repeats a condition forever

- `for` loop
	- loops for a certain amount of time

- `do...while`
	- does something while a condition is true
### Working With Different Data Types
##### Representing Different Data Types

- `%i` — integers
- `%s` — strings
- `%c` — characters
- `%f` — floating point values
- `%li` — long integers
##### Type Casting

- Converting one data type to another
##### Some Memory Problems We Might Run Into

- **Overflow** — when we don't have enough bits in memory to store our values
- **Floating-point Imprecision** — the boundary around how precise our values can be
