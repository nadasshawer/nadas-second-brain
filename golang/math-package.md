## The "math" Package

A built-in Go package that can be used to perform advanced mathematical operations
#### Syntax

- Use `math.` followed by the operation's name
- **Example** → `math.Min(2, 5)` to get the minimum value
#### The Power Function

- Acts the same as the exponent operator in JS (`**`)
- **Syntax** → `math.Pow(base, power)`
	- `fmt.Println(math.Pow(4, 2))` → output is `16`
